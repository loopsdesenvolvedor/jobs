# Jobs - Site de Empregos

Este é o repositório oficial do projeto "Jobs", um site de empregos desenvolvido com TypeScript, Next.js, Styled-Components e integrado com MySQL para o armazenamento de dados.

## UI/UX
O projeto está em desenvolvimento. Você pode conferir sua evolução através do [Figma](https://www.figma.com/design/o0NF1UaJRxLkJYn85h1ehR/jobs?node-id=0-1&t=f4Dkb94hddqxLTjk-0).

## Tecnologias Utilizadas

- **TypeScript**: Linguagem de programação utilizada para desenvolver o frontend e o backend do projeto.
- **Next.js**: Framework React para renderização de páginas do lado do servidor (SSR) e geração de páginas estáticas.
- **Styled-Components**: Biblioteca para estilização com CSS-in-JS, oferecendo componentes estilizados de forma eficiente.
- **MySQL**: Sistema de gerenciamento de banco de dados relacional utilizado para armazenar informações dos usuários e vagas de emprego.

## Estrutura do Projeto

O projeto está estruturado da seguinte forma:

- `/pages`: Contém as páginas do Next.js, cada arquivo corresponde a uma rota da aplicação.
- `/components`: Componentes React reutilizáveis.
- `/styles`: Estilos globais e estilizações utilizando Styled-Components.
- `/api`: Endpoints da API REST desenvolvida com Next.js API Routes para interação com o banco de dados.
- `/utils`: Funções utilitárias diversas.
