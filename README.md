# Jobs - Plataforma de Vagas de Emprego

O **Jobs** é uma plataforma online que conecta usuários em busca de oportunidades de emprego com empresas que desejam cadastrar vagas. Desenvolvido com **Express**, **Prisma**, **MySQL**, **EJS**, **JWT Token**, **Multer** e **TypeScript**, o site oferece funcionalidades dinâmicas e seguras para candidatos e empresas.

## Funcionalidades

- **Busca de Vagas:** Os usuários podem pesquisar por vagas de emprego filtrando por categoria, localização e outros critérios.
- **Cadastro de Vagas:** Empresas podem facilmente cadastrar novas vagas com todos os detalhes necessários, incluindo requisitos e benefícios.
- **Autenticação e Autorização:** A plataforma utiliza **JWT Tokens** para garantir segurança nas sessões de usuário e empresa.
- **Upload de Arquivos:** Usando **Multer**, o site permite que empresas façam o upload de arquivos, como descrições ou imagens de vagas.

## Tecnologias Utilizadas

- **Express:** Framework web para Node.js que gerencia as rotas e as requisições HTTP.
- **Prisma:** ORM para interação eficiente com o banco de dados **MySQL**.
- **MySQL:** Banco de dados relacional utilizado para armazenar informações de vagas e usuários.
- **EJS:** Motor de templates para renderização de páginas HTML dinâmicas.
- **JWT Token:** Sistema de autenticação segura usando JSON Web Tokens.
- **Multer:** Middleware para upload de arquivos em formulários.
- **TypeScript:** Garantindo um desenvolvimento mais seguro e robusto com tipagem estática.
